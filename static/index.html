Customer.io Build Flow

  1. Pick one base recipe from this page (do not start from scratch).
  2. Change copy, images, links and legal locale first; keep structure stable.
  3. Use one primary CTA and max one secondary CTA.
  4. Validate light and dark previews, then mobile.
  5. Only create new modules in library if no existing one fits.
