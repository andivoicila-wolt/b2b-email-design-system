Corner-Radius Rules

  1. Single card: border-radius="12px".
  2. Stacked card top: 12px 12px 0 0.
  3. Stacked card middle: no radius.
  4. Stacked card bottom: 0 0 12px 12px.
  5. Never combine full-radius siblings with the same background directly on top of each other.
  6. Internal media corners: standard image radius 10px.

Customer.io Build Flow

  1. Pick one base recipe from the category matrix above (do not start from scratch).
  2. Swap only module content first (copy, image, CTA target) before changing structure.
  3. Keep component order stable unless campaign goal requires a change in narrative flow.
  4. Use one primary CTA and one secondary CTA max per email to protect click clarity.
  5. Validate in preview for dark/light and mobile before copying final HTML into Customer.io.
  6. For new use-cases, add modules to Library first, then promote to a named template.
