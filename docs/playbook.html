Quick Build SOP (5 Steps)

  1. Pick one recipe row from the category matrix below.
  2. Open the corresponding template preview and duplicate that HTML as your base.
  3. Replace content slots only: headline, copy, image, CTA URL, legal/footer locale.
  4. Run preview in both themes and mobile width before exporting to Customer.io.
  5. If a needed block does not exist, add it to the component library first, then reuse it.

Non-Negotiable Design Rules

  1. Omnes with alternates active everywhere: ss01 + ss05 + ss07.
  2. Email width fixed at 640px; no custom widths per campaign.
  3. One primary CTA style across all modules; no ad-hoc button variants.
  4. Card radius logic: top 12 12 0 0, middle none, bottom 0 0 12 12, single 12.
  5. Inter-card spacing: 16px; major section break: 24px.
